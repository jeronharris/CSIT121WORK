Reflection on Best Design Practices

1. Mobile-first design ensures that the layout is user-friendly across devices. Simplifying the content into a single-column format for small screens improves readability.

2. Using CSS media queries, the layout can be adjusted to utilize more screen space on larger devices, such as switching to multiple columns on desktop screens.

3. Consistent navigation and anchor links help users quickly navigate between sections of the page, improving user experience on both mobile and desktop views.

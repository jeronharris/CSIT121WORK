What I Learned

In this lab, I learned about common HTML and CSS form elements, like input fields, buttons, and dropdown menus, and how to use them to create interactive forms. I also learned how to create and style tables and forms on web pages, making sure they look good and are easy to use. I also explored ways to improve the accessibility of tables and forms by adjusting their structure and applying CSS styling.
